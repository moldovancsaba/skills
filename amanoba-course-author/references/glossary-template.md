# Glossary Template

Use this template to lock terminology for a target language.

## Format

For each term, include:

- `Term (source)`
- `Approved translation`
- `Forbidden translations`
- `Context or example`
- `Notes`

## Example entry

- `Agent`
- `Agente`
- `Asistente`, `Bot`
- `Build an agent that can execute a workflow without manual prompts.`
- `Use Agente for this course, avoid Bot.`
